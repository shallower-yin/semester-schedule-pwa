安装说明

网页版（电脑 / 手机浏览器）

  1. 点击「安装 / 打开网页版」进入应用。
  2. 在 Edge / Chrome 菜单中选择「安装应用」或「添加到主屏幕」。
  3. 登录账号后即可同步日程数据。

Android APK

  1. 点击「下载 Android APK」，等待下载完成。
  2. 用系统文件管理打开安装包；若提示未知来源，请允许本次安装。
  3. 同签名覆盖安装可保留本地数据；首次使用建议允许通知权限。
  4. 登录同一账号并同步，与电脑网页版共用数据。

国内网络可优先使用镜像下载。若 GitHub Pages 打不开，可打开 镜像分发页。
